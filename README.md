#📊 Bank Customer Insights Dashboard#

1. 🏦 Bank Customer Insights Dashboard: Credit Score, Customer Exit & Demographic Analysis

An interactive Power BI dashboard that analyes bank customer data to understand credit scores, customer exits, and customer demographics such as age, gender, and country.

2. The Bank Customer Insights Dashboard is a Power BI report that helps analyze bank customers based on their credit score, age group, gender, and location.

The purpose of this dashboard is to understand customer behavior, customer exit patterns, and customer distribution across countries so banks can make better decisions to improve customer retention.

3. The dashboard was built using the following tools and technologies:

• 📊 Microsoft Power BI Desktop – Used to create the dashboard and visualizations.
• 📂 Power Query – Used to clean and prepare the data.
• 🧠 DAX (Data Analysis Expressions) – Used to create calculated measures and perform analysis.
• 🧩 Data Modelling – Used to organise and connect the data tables.
• 📁 File Format – .pbix file for development and .png images for dashboard preview.

4. Data Source

Source: Dataset downloaded from Maven Analytics.

The dataset contains bank customer information such as:

- Customer age group (Adult and Senior)
- Gender (Male and Female)
- Country (France, Germany, Spain)
- Credit score categories (Good, Fair, Worst, Excellent)
- Customer tenure
- Account balance
- Customer exit status (Exited or Not Exited)

This data helps analyze customer behaviour and bank customer retention.

5. Business Problem

Banks often find it difficult to understand why customers leave the bank.

Without proper analysis, it is hard to know:

- Which customers are likely to leave
- Which countries have more customers
- How credit score affects customer behaviour
- Which customer groups are most valuable
- Which customer is using less service

6. Goal of the Dashboard

The main goal of this dashboard is to:

- Analyze customers who exited and customers who stayed
- Understand credit score distribution
- Analyze customer demographics (age and gender)
- Compare customer data by country
- Provide insights that help banks improve customer retention
